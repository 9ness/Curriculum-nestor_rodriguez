# Startups de IA en España para Backend Senior Java + IA (remoto, Galicia)

*Julio 2026. Perfil: Backend Senior Java/Spring Boot + microservicios + integración LLMs (OpenAI/Claude/Gemini/DeepSeek) + agentes + VPS. Inglés intermedio. Objetivo 40-55k. Remoto.*

## Aviso de realidad
1. **Java no es el stack dominante en IA española** — predomina Python/Node. Tu venta: "backend senior de microservicios que integra LLMs", agnóstico de stack.
2. **El inglés es el mayor filtro**, no el sueldo. Startups con inversores/clientes internacionales piden inglés medio-alto casi siempre.
3. Producto para **mercado español** (voz/atención cliente, legal ES, fintech ES, salud ES) = menos inglés.
